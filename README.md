## How to Use:


### Prerequisites
- Make sure you have Node.js installed on your machine.
- Install Expo CLI

### <a>1.Clone the Repository:</a>
    git clone https://github.com/sergioaguilar13y19/Weather-App
### <a>2.Navigate to the Project:</a>
    cd Weather-App
### <a>3.Install Dependencies</a>
    npm install
### <a>4.Run App</a>
    npx expo start
